# Number Guessing Game

## Description
A simple Python command-line game where the computer randomly selects a number between 1 and 100. The player guesses the number, and the program gives hints ("Too High" or "Too Low") until the correct number is guessed.

## Features
- Random number generation
- Higher/Lower hints
- Counts the number of attempts
- Simple command-line interface

## Technologies Used
- Python 3
- random module

## How to Run
1. Open the project folder.
2. Run:
   python guess_game.py
3. Enter your guesses until you find the correct number.
